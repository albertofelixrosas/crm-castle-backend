# Contexto y Objetivo

## Contexto

Se busca construir una API REST profesional en Java 21 con Spring Boot para simular un flujo de CRM inspirado en Salesforce.

## Objetivo principal

Implementar un mini CRM donde:

1. Se crean Leads.
2. Se actualizan Leads.
3. Se convierten Leads a Contact.
4. La conversion crea o reutiliza Company.
5. El Lead queda marcado como CONVERTED.

## Alcance inicial

- Dominio principal: Lead, Contact, Company.
- Enfoque: backend portfolio-ready para nivel junior/intermedio.
- Prioridad: claridad arquitectonica, consistencia REST, reglas de negocio claras y mantenibilidad.
